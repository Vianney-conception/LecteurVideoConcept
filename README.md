# Lecteur vidéo CSS et JS

---

## Présentation

Ce projet propose un lecteur vidéo moderne réalisé principalement avec **JavaScript**, **HTML**, **CSS**.

---

## Fonctionnalités

- Lecture/Pause/Stop de la vidéo
- Barre de progression stylisée en CSS, avec miniatures.
- Interface personnalisable
- Contrôle avancé via JS : saut, etc.

---

## Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Vianney-conception/LecteurVideoConcept.git
   ```
2. Ouvrez `index.html` dans votre navigateur.
---

## Utilisation

- Placez vos vidéos dans le dossier du projet.
- Personnalisez l’apparence en modifiant les fichiers CSS.
- Profitez du lecteur en lançant `index.html`.
